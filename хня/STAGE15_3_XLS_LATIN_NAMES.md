# STAGE15.3 - XLS Latin Names

Исправление по просьбе: все Excel-файлы внутри проекта и все автоматически создаваемые `.xlsx`-результаты теперь имеют латинские имена.

Что изменено:

- убрана ссылка на кириллический шаблон `подсчёт_образец.xlsx`;
- авто-поиск шаблона теперь ищет только латинские имена в `Samples`;
- добавлен латинский файл шаблона `Samples/vote_counter_template.xlsx`;
- генератор итогового протокола транслитерирует название конкурса в имя файла;
- если название пустое, результат сохраняется как `Contest.xlsx`;
- архив проверен: внутри нет `.xls/.xlsx/.xlsm/.xlsb` с кириллицей в имени.

Примеры:

```text
Любимый город (12) -> Lyubimyy gorod(12).xlsx
Конкурс -> Konkurs.xlsx
```
