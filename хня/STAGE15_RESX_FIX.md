# STAGE15.1 - RESX FIX

Исправление для ошибки Visual Studio/MSBuild:

```text
Не удается найти файл ресурсов: "**/*.resx".
```

Что изменено:

- отключен автоматический поиск `.resx` через SDK glob;
- `MainForm.cs` явно помечен как WinForms Form;
- `MainForm.resx` оставлен как обычный файл, чтобы не ломать старые копии проекта, но не передается в `GenerateResource`;
- версия поднята до `1.14.1-stage15-firebird-import-spellcheck-resx-fix`.

Причина: на некоторых конфигурациях VS/MSBuild шаблон `**/*.resx` может попасть в задачу генерации ресурсов как буквальная строка. Для текущей формы ресурсы дизайнера не используются, поэтому безопаснее отключить автоматическую обработку `.resx`.
