# STAGE31 - Author Import Quick Access

Добавлено более заметное принятие раскрытия авторов.

## Что изменилось

- На первой вкладке **Конкурсы** добавлена кнопка **Импорт авторов** рядом с импортом конкурса из текста.
- На вкладке **Голосование** область ввода теперь подписана как **Вставь голосование или раскрытие авторов сюда**.
- На вкладке **Голосование** добавлена кнопка **Принять авторов**.
- В popup-меню грида голосов добавлена команда **Принять авторов из текста**.
- Импорт авторов можно делать прямо из текста, вставленного в основное поле голосования.
- Если основное поле пустое, откроется старый диалог импорта авторов с подхватом текста из буфера.

## Поддерживаемый формат

```text
001, 004, 008 - Лидия Андрианова (Медведева)
002 - Марина Аллахвердова
003 - Татьяна Розина
```

## База

Рабочая база остаётся в `VoteCounter/Database/VoteCounter.db` и не кладётся в ZIP.
