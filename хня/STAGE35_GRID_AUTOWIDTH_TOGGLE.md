# STAGE35 - Grid AutoWidth Toggle

Добавлено штатное DevExpress-растягивание колонок грида на всю ширину рабочей области.

## Что изменено

- В popup-меню каждого `GridView` добавлен пункт `AutoWidth на весь экран`.
- Используется родное свойство DevExpress:

```csharp
view.OptionsView.ColumnAutoWidth = true;
```

- Старый пункт автоширины переименован в `Автоширина по содержимому`, чтобы было понятно различие:
  - `Автоширина по содержимому` - подгоняет колонки по тексту;
  - `AutoWidth на весь экран` - растягивает колонки на всю ширину грида.
- Состояние `ColumnAutoWidth` сохраняется в layout внутри базы, в `Setting.Ini`.
- Рабочая база остаётся в `VoteCounter/Database/VoteCounter.db` и не включается в ZIP.
