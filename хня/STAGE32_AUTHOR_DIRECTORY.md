# STAGE32 - Справочник авторов

Добавлена отдельная вкладка **Авторы**.

Что есть:

- общий список авторов из базы и из работ конкурсов;
- количество работ автора;
- количество конкурсов;
- сумма принятых баллов;
- средняя оценка по работам;
- список конкурсов и работ автора;
- быстрые команды: показать работы, найти работы автора в текущем конкурсе, добавить автора в обязательные судьи, скопировать карточку автора.

Справочник пополняется после импорта раскрытия авторов и после сохранения работ с заполненным полем `Автор`.

Рабочая база по-прежнему остаётся в `VoteCounter/Database/VoteCounter.db` и не кладётся в ZIP.
