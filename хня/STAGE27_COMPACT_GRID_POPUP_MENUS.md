# STAGE27 - Compact Grid Popup Menus

Контекстные меню таблиц оставлены, но сделаны компактными.

## Что изменено

- В верхнем уровне popup-меню теперь остаются только главные команды конкретной таблицы.
- Если у таблицы много команд, первые три показываются сразу, остальные уходят в подменю `Ещё команды...`.
- Общие команды таблицы сжаты до трёх быстрых пунктов:
  - `Копировать строку`;
  - `Экспорт CSV...`;
  - `Автоширина колонок`.
- Остальное перенесено в подменю `Ещё...`:
  - копировать ячейку;
  - копировать выбранные строки;
  - копировать все видимые строки;
  - сбросить фильтр;
  - сбросить сортировку;
  - сбросить группировку;
  - развернуть / свернуть группы;
  - сохранить / сбросить layout всех таблиц.

## Зачем

STAGE26 сделал все команды доступными, но меню получилось перегруженным.
STAGE27 сохраняет все возможности, но убирает визуальный шум.

## Важно

- Рабочая база `VoteCounter/Database/VoteCounter.db` не включается в ZIP.
- Layout по-прежнему хранится в таблице `Setting.Ini` внутри рабочей базы.
- Правило `3+ = 3.5` сохранено.
