# STAGE26 - Grid Popup Menus

Добавлен единый слой контекстных меню для всех DevExpress GridControl/GridView.

## Что появилось

Правый клик по любой таблице открывает popup-меню. В верхней части меню находятся команды, относящиеся именно к этой таблице, ниже - общие команды таблиц.

## Табличные команды

### Конкурсы
- открыть выбранный конкурс;
- перейти в настройки;
- перейти в итоги;
- создать конкурс;
- импортировать конкурс из текста;
- импортировать старую *.fdb;
- обновить список;
- удалить конкурс.

### Работы / рейтинг
- открыть конкурс;
- перейти в итоги;
- обновить карточку;
- сохранить пакет отчётов.

### Голосующие / контроль
- перейти в контроль голосования;
- обновить карточку;
- скопировать должников;
- добавить неизвестных в судьи.

### Голоса конкурса
- проверить текст;
- принять в базу;
- выполнить авточек правил;
- показать базу конкурса;
- очистить голоса конкурса;
- сформировать Excel.

### Работы конкурса
- добавить работу;
- удалить выбранные работы;
- добавить номера работ из голосов;
- импортировать работы из текста;
- принять одну работу;
- сохранить настройки конкурса.

### Голосующие / судьи
- добавить голосующего;
- добавить голосующих из базы;
- добавить авторов в судьи;
- сохранить настройки конкурса.

### Контроль голосования
- обновить контроль;
- скопировать должников;
- сохранить отчёт контроля;
- добавить неизвестных в судьи;
- сохранить настройки.

### Итоги
- обновить итоги;
- скопировать протокол;
- скопировать победителей;
- сформировать Excel;
- сохранить пакет отчётов;
- HTML-протокол;
- печать HTML;
- открыть отчёты.

### Правки правил
- обновить правки;
- скопировать правки;
- сохранить CSV.

## Общие команды для всех таблиц

- копировать ячейку;
- копировать строку;
- копировать выбранные строки;
- копировать все видимые строки;
- экспорт видимых строк в CSV;
- автоширина колонок;
- сброс фильтра;
- сброс сортировки;
- сброс группировки;
- развернуть группы;
- свернуть группы;
- сохранить layout всех таблиц;
- сбросить layout всех таблиц.

## Важно

Рабочая база `VoteCounter/Database/VoteCounter.db` в архив не включается и не затирается.
