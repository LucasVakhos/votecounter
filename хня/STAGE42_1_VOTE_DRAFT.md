# Stage 42.1 - черновик голосования

Добавлен первый практический шаг Stage 42.

Что сделано:
- на панели голосования добавлены кнопки `Сохранить черновик`, `Загрузить черновик`, `Очистить черновик`;
- черновик хранится в `AppSettings` внутри SQLite-базы;
- сохраняется текст черновика, конкурс и дата обновления;
- при загрузке черновика от другого конкурса показывается подтверждение;
- при загрузке поверх непустого поля показывается подтверждение;
- очистка черновика не трогает принятые голоса.

Технически:
- добавлены поля `VoteDraftContestId`, `VoteDraftText`, `VoteDraftUpdatedAt` в `AppSettings`;
- сохранение идёт через существующую таблицу `AppSettings`, отдельная миграция не нужна;
- вложенная папка `votecounter/**` исключена из компиляции, чтобы случайная копия проекта не ломала сборку дублями типов.

Проверка:
- `dotnet build --no-restore -p:RestorePackagesPath="C:\Users\User\.nuget\packages" -p:NuGetAudit=false`
- результат: 0 ошибок, 0 предупреждений.
