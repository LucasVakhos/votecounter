# STAGE21 - HTML Export / Print Pack

Добавлен красивый пакет экспорта итогов конкурса без переноса рабочей базы в архив.

## Что добавлено

- На вкладке **Итоги** появились кнопки:
  - **HTML-протокол** - сохраняет пакет отчётов и открывает красивый HTML;
  - **Печать HTML** - открывает печатную HTML-версию с автоматическим вызовом окна печати браузера.
- В отчётный пакет теперь дополнительно входят:
  - `*_protocol.html`;
  - `*_print.html`.
- HTML-протокол содержит:
  - карточки статистики конкурса;
  - готовый итоговый протокол;
  - рейтинг работ;
  - контроль голосующих;
  - таблицу всех голосов.
- Печатная версия имеет CSS `@media print` и не требует Excel.
- Рабочая база **не кладётся в ZIP** и не должна затираться при распаковке поверх проекта.

## База

Рабочая база остаётся на прежнем месте:

```text
VoteCounter/Database/VoteCounter.db
```

В архиве её быть не должно.
