# STAGE22 - Vote Rules + Decimal Score Fix

Что изменено:

- импорт голосования автоматически восстанавливает правила конкурса из текста, если в базе они ещё не заполнены;
- распознаётся правило `Разрешается голосовать оценками: 1, 2, 3, 4`;
- распознаётся правило `высший балл дается только одной работе в одном задании`;
- включается режим `одна 4 в теме`, лишняя 4 понижается до базовой оценки;
- самоголосование включается как `0`, если в тексте есть упоминание голосов за свои работы;
- `3+` теперь считается как `3.5`;
- `Score` и `OriginalScore` переведены на `decimal` в моделях и `REAL` в новой SQLite-схеме;
- итоги, средний балл, CSV/HTML/Excel/DevExpress Grid теперь получают дробные значения без обрезания.

База остаётся на прежнем месте:

```text
VoteCounter/Database/VoteCounter.db
```

Рабочая база не кладётся внутрь ZIP, чтобы чекпойнты не затирали данные при распаковке поверх проекта.
