# STAGE23 - Vote Import Preview Guard

Добавлен безопасный слой перед импортом голосов.

- перед записью показывается подтверждение: сколько новых голосующих, сколько будет заменено, сколько старых голосов затронет импорт;
- перед записью сохраняется отчёт предпросмотра в `Database/Reports/yyyyMMdd_HHmmss_vote_import_<contest>/`;
- добавлена кнопка `Отчёт проверки`;
- отчёт содержит `import_preview.txt`, `incoming_votes.csv`, `replaced_voters.csv`, `warnings.txt`;
- правило `3+ = 3.5` сохранено;
- рабочая база `Database/VoteCounter.db` в ZIP не включается.


> Примечание: начиная со STAGE24 подтверждение перед записью заменено на автоматический чек правил.
