# STAGE24 - Vote Rule Autocheck

Изменение по требованию: вместо подтверждения перед записью включён автоматический чек по правилам голосования.

## Что изменено

- Кнопка `Отчёт проверки` переименована в `Авточек правил`.
- `Принять в базу` больше не спрашивает подтверждение перед записью.
- Перед сохранением голосов правила применяются автоматически.
- Если лимит максимальной оценки `4` исчерпан, то в расчётное поле `Score` принимается базовая оценка, обычно `3`.
- Исходная оценка не теряется: она остаётся в `OriginalScore` и `OriginalScoreText`.
- Для `3+` расчётное значение остаётся `3.5`.
- Отчёт авточека сохраняется в `Database/Reports`, но если отчёт не удалось записать, импорт голосов не блокируется.

## Пример

Если по теме разрешена одна `4`, а голосующий поставил две `4`, то первая остаётся `4`, а следующая сохраняется так:

```text
OriginalScoreText = 4
OriginalScore     = 4
ScoreText         = 3
Score             = 3
RuleNote          = лимит 4 исчерпан: принято 3, оригинал сохранён
```
