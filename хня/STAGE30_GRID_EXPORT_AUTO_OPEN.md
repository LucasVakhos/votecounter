# STAGE30 - Grid Export Auto Open

Добавлено автоматическое открытие документа после экспорта из popup-меню DevExpress Grid.

## Что изменено

- Команда `Экспорт CSV...` в popup-меню любого грида теперь после сохранения сразу открывает созданный файл системной программой.
- Для `.csv` обычно откроется Excel или программа, назначенная в Windows по умолчанию.
- Если открыть файл автоматически не получилось, программа не падает: файл остаётся сохранённым, а пользователь видит предупреждение с путём к файлу.

## База

Рабочая база не входит в ZIP и не затирается при распаковке:

```text
VoteCounter/Database/VoteCounter.db
```
