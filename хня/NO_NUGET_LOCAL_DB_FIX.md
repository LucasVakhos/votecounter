# Историческая заметка: NO NUGET LOCAL DB FIX

В одном из ранних чекпойнтов проект временно был возвращён на JSON-хранение,
потому что тогда требовалось собрать приложение без NuGet и без SQLite.

В STAGE14 это ограничение отменено требованием: **сделать хранение в базе**.

Актуальное состояние STAGE15:

- используется `Microsoft.Data.Sqlite`;
- используется `FirebirdSql.Data.FirebirdClient` для импорта старых `*.fdb`;
- база создаётся в папке проекта: `VoteCounter/Database/VoteCounter.db`;
- JSON-миграция убрана;
- старые конкурсы импортируются из Firebird через кнопку **Импорт старой *.fdb**.
