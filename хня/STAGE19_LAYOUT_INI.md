# STAGE19 - Layout INI рядом с exe

Добавлено сохранение раскладки интерфейса в обычный INI-файл рядом с исполняемым файлом:

```text
VoteCounter.layouts.ini
```

## Что сохраняется

- положение и размер главного окна;
- состояние окна: normal / maximized;
- выбранная вкладка;
- положение split-панелей;
- ширина, порядок, видимость и FillWeight колонок всех `DataGridView`;
- последний выбранный конкурс.

## Как работает

- При запуске программа читает `VoteCounter.layouts.ini`, если файл существует.
- При закрытии программа автоматически сохраняет текущую раскладку.
- На вкладке `Конкурсы` добавлены кнопки:
  - `Сохранить layout` - сохранить вручную;
  - `Сбросить layout` - удалить INI и вернуться к стандартной раскладке при следующем запуске.

## Важно

INI хранится именно рядом с exe, а не в `%LocalAppData%` и не в базе.
База конкурса по-прежнему лежит в папке проекта:

```text
VoteCounter/Database/VoteCounter.db
```

JSON-хранилище не возвращалось.
