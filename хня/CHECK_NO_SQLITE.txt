STAGE15 NOTE

Предыдущее ограничение "без SQLite / без NuGet" снято по новому требованию:
хранение переведено в базу данных.

Теперь используется:
- Microsoft.Data.Sqlite для рабочей базы проекта;
- FirebirdSql.Data.FirebirdClient для импорта старых *.fdb;
- локальный файл базы VoteCounter/Database/VoteCounter.db.

JSON-миграция убрана. Старые конкурсы импортируются через кнопку
"Импорт старой *.fdb".
