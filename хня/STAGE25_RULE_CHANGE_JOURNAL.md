# STAGE25 - Rule Change Journal

Добавлена отдельная вкладка **"Правки правил"**.

Что умеет:

- показывает все голоса, которые были приняты иначе по правилам конкурса;
- сохраняет оригинальную оценку отдельно от расчётной;
- показывает причину: лимит максимальной оценки, самоголос, дробная оценка `3+ = 3.5`, прочие автоправки;
- подсвечивает строки:
  - самоголосы - красноватым;
  - лимит максимальной оценки - жёлтым;
  - `3+ = 3.5` - голубым;
- копирует журнал правок в буфер обмена;
- сохраняет CSV в `VoteCounter/Database/Reports/...`;
- рабочая база `VoteCounter.db` по-прежнему не кладётся в ZIP.
