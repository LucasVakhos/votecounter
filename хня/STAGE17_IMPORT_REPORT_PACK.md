# STAGE17 - Import Report Pack

Цель этапа - после импорта старых Firebird-баз `*.fdb` сразу получать проверяемый пакет отчётов, а не только сообщение об успешной операции.

## Что добавлено

- Автоматическое создание папки `Database/Reports/yyyyMMdd_HHmmss_firebird_import` после импорта.
- `import_summary.txt` с параметрами импорта, backup-путём, количеством добавленных/обновлённых конкурсов, голосов и предупреждений.
- Для каждого выбранного конкурса:
  - `*_final_protocol.txt`;
  - `*_rating.csv`;
  - `*_vote_control.csv`;
  - `*_votes.csv`.
- Кнопка `Сохранить отчёт контроля` на вкладке `Контроль голосования`.
- Кнопки `Сохранить пакет отчётов` и `Открыть отчёты` на вкладке `Итоги`.
- Латинские имена файлов отчётов через встроенную транслитерацию.

## Где лежит

```text
VoteCounter/Database/Reports/
```
