# STAGE20.3 - Contest Text Syntax Fix

Исправлен синтаксический сбой в `ContestTextImporter.cs`, появившийся после патча OK-импорта.

## Что исправлено

- Убраны случайно вставленные regex-строки из тела метода `DetectContestName`.
- Восстановлены проверки `if (match.Success)` для:
  - строки объявления конкурса с названием в кавычках;
  - отдельной строки-названия в кавычках.
- Логика распознавания конкурса сохранена:
  - `178й ... конкурс "Нарисуй мне, художник, картину!"` -> номер `178`;
  - название -> `Нарисуй мне, художник, картину!`;
  - даты OK больше не становятся работами.
