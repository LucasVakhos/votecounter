# STAGE20.2 - Contest Text OK Import Fix

Исправлен импорт конкурса из длинного текста ОК/форумов.

## Что исправлено

- Номер конкурса теперь распознаётся не только в формате `конкурс 178`, но и в формате `178й 2х недельный стихотворный конкурс ...`.
- Название конкурса теперь в первую очередь берётся из кавычек в строке объявления конкурса: `конкурс "Нарисуй мне, художник, картину!"`.
- Служебные строки вроде `УВАЖАЕМЫЕ АВТОРЫ!`, `ЗАДАНИЯ КОНКУРСА`, `Ответить`, дат `10 июн` больше не становятся названием или работами.
- Добавлено отдельное распознавание блоков ОК:

```text
001
Художник
Александр Кремер
" Июнь"
( Картина )
.
текст работы...
```

- Для таких блоков название работы берётся из строки в кавычках, автор работы остаётся `Неизвестный автор`, а художник и картина сохраняются в `Topic`.
- Полный текст работы сохраняется в `ContestWork.Content`.

## Проверочный пример

Строка:

```text
178й 2х недельный стихотворный конкурс "Нарисуй мне, художник, картину!" начинается!
```

должна дать:

```text
№ конкурса: 178
Название: Нарисуй мне, художник, картину!
```
