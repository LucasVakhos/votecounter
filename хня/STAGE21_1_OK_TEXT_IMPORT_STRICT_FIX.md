# STAGE21.1 - OK text import strict fix

Исправлен импорт конкурса из большой вставки Одноклассников/форумной ленты.

## Что исправлено

- Текст вида `178й ... конкурс "Нарисуй мне, художник, картину!"` корректно даёт конкурс №178 и название из кавычек.
- Для ОК-экспорта включён строгий режим распознавания работ: берутся только блоки вида `001 / Художник / Автор картины / "Название картины"`.
- Служебные строки `Ответить`, даты `10 июн`, счётчики реакций `1`, `2`, `3`, `4` больше не превращаются в работы и не затирают реальные номера.
- Строки профиля ведущего после реакции больше не попадают в авторы или названия работ.
- `Ведущая Елька )))` теперь распознаётся как ведущий даже без двоеточия.
- Работы 003 и 004 больше не пропадают после импорта такого текста.
