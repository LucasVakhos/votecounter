# Stage 40.3 - импорт работ из лички

Сделан отдельный сценарий для кнопки "Импорт работ из лички" в верхней панели управления конкурсом.

Что умеет:
- вставка текста из буфера;
- разбор блоков вида "Автор: ..." + "Название: ..." + текст;
- разбор простого блока: автор, название, текст;
- несколько работ можно разделять строкой `---`;
- автоматическая нумерация с выбранного номера;
- предпросмотр номера, автора, названия и статуса;
- проверка работ перед сохранением;
- безопасное дополнение списка без перезаписи существующих номеров по умолчанию.

Старый импорт работ/авторов из текста оставлен отдельно для шаблонов, списков и раскрытия авторства.
