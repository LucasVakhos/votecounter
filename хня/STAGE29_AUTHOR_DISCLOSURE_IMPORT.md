# STAGE29 - Author Disclosure Import

Добавлен отдельный импорт раскрытия авторства после закрытия конкурса.

## Поддерживаемый формат

```text
В конкурсе приняли участие 6 авторов ( 9 стихотворений):

001, 004, 008 - Лидия Андрианова (Медведева)
002 - Марина Аллахвердова
003 - Татьяна Розина
005 - Лаврентий Беспутнов
006 - Елена Себова
007, 009 - Роман Рыбак
```

## Что делает импорт

- сопоставляет авторов с существующими работами по номеру;
- обновляет поле `ContestWorks.Author`;
- пополняет справочник `Authors` в SQLite;
- создаёт связь `ContestWorkAuthors` между работой и автором;
- по галочке добавляет авторов в обязательные судьи;
- предупреждает, если в раскрытии есть номер работы, которого нет в конкурсе;
- проверяет заявленное количество авторов и работ из первой строки.

## Новые таблицы

```sql
Authors(NormalizedKey, Name, CreatedAt, UpdatedAt)
ContestWorkAuthors(ContestId, WorkNo, AuthorKey, AuthorName, UpdatedAt)
```

Рабочая база остаётся на прежнем месте:

```text
VoteCounter/Database/VoteCounter.db
```

`VoteCounter.db` не кладётся в ZIP, чтобы не затирать рабочие данные.
