# STAGE39.2 - Заполнение списка текущего конкурса без DataSource

Что исправлено:

- верхний список `Текущий конкурс` теперь наполняется вручную через `Items`;
- привязка `DataSource` убрана из рабочего наполнения списка;
- если после обновления панели список пустой, но конкурсы загружены, панель заполняет список повторно;
- если выбранный конкурс не найден, автоматически выбирается первый доступный пункт;
- строка списка остаётся читаемой: `№ - Название (активный/архив)`.

Проверка:

- `dotnet build --no-restore -p:RestorePackagesPath="C:\Users\User\.nuget\packages" -p:NuGetAudit=false`
- результат: 0 ошибок, 0 предупреждений.
