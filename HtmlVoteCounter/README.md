# VoteCounter HTML

Браузерная версия основных сценариев WinForms-приложения VoteCounter.

## Запуск

Откройте `HtmlVoteCounter/index.html` в браузере. Сервер не нужен.

## Что есть

- список конкурсов;
- карточка конкурса;
- импорт работ из текста;
- раскрытие авторов после закрытия;
- разбор голосований из текста;
- применение основных правил голосования;
- контроль пропусков и неизвестных номеров;
- рейтинг и итоговый текст;
- журнал правок правил;
- список участников;
- экспорт JSON, CSV и HTML.

## Хранение

Данные сохраняются в `localStorage` браузера. Для переноса или резервной копии используйте кнопку `JSON` в верхней панели.

## Отличия от WinForms

HTML-версия не подключается напрямую к SQLite, Firebird, DevExpress и Excel. Вместо этого она работает автономно в браузере и экспортирует данные в файлы.
